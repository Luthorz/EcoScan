♻️ EcoScan

Aplicação desenvolvida em Python e Streamlit para identificação de resíduos recicláveis.

Tecnologias:
- Python
- Streamlit
- TensorFlow
- OpenCV
- Pillow
- NumPy
- gTTS

Objetivo:
Auxiliar na separação correta de resíduos e promover a educação ambiental através da classificação automática de imagens.
